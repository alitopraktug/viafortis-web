# ViaFortis Website

Production website developed for a UK-based carpentry and aluminium fencing company.

## 🌐 Live Website

https://www.viafortis.co.uk

## Overview

ViaFortis is a modern business website built for a UK-based company specializing in bespoke carpentry, aluminium fencing, gates, and outdoor solutions.

## Features

- Responsive Design
- Product Catalog
- Shopping Cart
- Stripe Payment Integration
- Contact Form
- SEO Optimized
- Mobile Friendly

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Stripe
- Framer Motion
- Vercel

## Installation

```bash
npm install
npm run dev
```
